Script de External MergeSort: Main.java
Desarrollado en:
Requiere Java version 7 o mayor.
IntelliJ IDEA 2017.1.2
JRE: 1.8.0_112-release-736-b16 x86_64
JVM: OpenJDK 64-Bit Server VM by JetBrains s.r.o
Mac OS X 10.11.6 

Para correr este script es necesario crear una carpeta en el mismo directorio donde se encuentre este y dentro de ella guardar todos los inputs. Si la carpeta tiene como nombre NombreCarpeta entonces el script se usa de la siguiente manera:

java Main B M NombreCarpeta CoordenadaAOrdenar
Ej: 
java Main 4000 40000 Inputs x

Asegurarse que los únicos archivos dentro de esta carpeta sean los inputs. Revisar y eliminar archivos ocultos de existir (.Ds_Store por ejemplo en OSX).
